Performance Weightage

In this chart avg. weightage of different scholastic exam type is calculated and shown in meter chart. There are four colour sections in this chart which indicate below 70% (red), 70 % to 80% yellow, 81% to 90% green, above 90% blue. The weightage is 10% for chapter test, 20% for module tests and 70% of mock tests. All the data from exam type will be in percentage. On click details on this chart a table will be display, which is consist of two row and three columns. In rows avg. percentage and weightage (After calculating the weightage given above). In column Ch test, Module and Mock.

Strength Analysis

This is basically the area chart of avg. percentage marks with respect to different subjects and different exam type. In x-axis the subjects will display and in y-axis the percentage will display. In this chart three different colour area will be used for three different exam type Ch test, module and mock. By disabling an exam type or a set of exams type we can view only a particular exam type performance. So, we can view in different combinations i.e – Ch test vs Module, Module vs Mock etc.


Performance Analysis

This is basically the column chart where in x-axis the subjects will display and y-axis percentage will display. By disabling an exam type or a set of exams type we can view only a particular exam type performance. There are three columns against each subject which is for three different exam type Ch test, module and mock. Three different colour combination in three columns. By clicking on chart details tab a table will display, in which rows are filled by different subjects and columns are filled by Ch test, Module and mock. All data in the mentioned chart will be in percentage.

Overall Performance

In this chart avg. of a competitive exam type is calculated and shown in meter chart. There are four colour sections in this chart which indicate below 70% (red), 70 % to 80% yellow, 81% to 90% green, above 90% blue. On click details on this chart a table will be display, In row the exam name and in column Set wise avg. percentage marks.








Average Score vs Market Trend

• This chart is the average score of different subjects among all sets. This chart is dynamic, as per the Set given the chart will differ.

• There is no redirection option in this chart, this chart is only helping student to display where they stand in a particular competitive exam with respect to other students.

• Exam name wise this chart may vary, i.e For NSTSE social science subjects will not display.




Score Comparison

• This Chart is the comparison of total score in different Sets.

• On click details a table will display where in columns exam subtype and total marks and in rows set name will display.

• On click tab (Set 1, Set 2 etc.) redirect to individual set score pie chart (Not attempted, correct and incorrect).

• On click details on pie chart a table will display where, there are four columns Total questions, Attempted, correct, not attempted and incorrect. In rows the specific exam subtype will be shown.
